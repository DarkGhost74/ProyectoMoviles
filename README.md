# Proyecto Dispositivos Móviles
Este es el PIA del equipo 7 de la brigada 2 del grupo 001 en la materia de ingeniería de dispositivos móviles.

## Arquitectura
El proyecto está construido sobre una plantilla en blanco de ReactNative con Expo. Todas las pantallas nuevas deben agregarse en la carpeta `screens`.
Asi como los componentes deberán agregarse dentro de la carpeta `components`.

## Instalación y ejecución
Para descargar el proyecto basta con hacer un `git clone` al repositorio.

Es necesario tener previamente instalado Node.js para poder ejecutar el proyecto.
Una vez instalado tanto el repositorio como Node.js basta con ejecutar
```
npm install
```
para poder instalar todas las dependencias necesarias.

Ya que se hayan seguido los pasos previos se puede empezar a escribir código. Para ejecutar el servidor de pruebas se debe de ejecutar el siguiente comando:
```
npx expo start
```
Una vez estando en la terminal donde se está corriendo el servidor se pueden escribir las letras `i` para correr el simulador de iOS, `a` para el de Android, y `w` para el web.
También se puede descargar la aplicación de `expo go` en Android o iOS para poder ejecutar en tiempo real la app ahí mismo ingresando la URL o escaneando el código QR que aparece en la terminal.
